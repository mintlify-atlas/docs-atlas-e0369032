This guide will walk you through creating your first application with the Lexa JS SDK, from installation to making your first API call.

## Installation

First, install the Lexa SDK in your project:

<CodeGroup>

```bash npm
npm install @robilabs/lexa
```

```bash yarn
yarn add @robilabs/lexa
```

```bash pnpm
pnpm add @robilabs/lexa
```

</CodeGroup>

## Setup Your API Key

Set up your Lexa API key as an environment variable:

```bash .env
LEXA_API_KEY=your_api_key_here
```

<Note>
Don't have an API key yet? Get one at [lexa.chat](https://www.lexa.chat) by signing up and navigating to your account settings.
</Note>

## Your First Chat Completion

Create a new file and add the following code to make your first chat completion request:

```javascript index.js
import Lexa from '@robilabs/lexa';

const lexa = new Lexa(process.env.LEXA_API_KEY);

const completion = await lexa.chat({
  messages: [
    { role: 'user', content: 'Hello! How are you?' }
  ],
  model: 'lexa-mml',
  temperature: 0.7,
  max_tokens: 100
});

console.log(completion.choices[0].message.content);
```

Run your code:

```bash
node index.js
```

You should see a friendly response from Lexa!

## Common Use Cases

### Basic Conversation

Create a simple chat interaction:

```javascript
const completion = await lexa.chat({
  messages: [
    { role: 'user', content: 'Can you tell me a joke?' }
  ],
  model: 'lexa-mml',
  temperature: 0.7,
  max_tokens: 100
});

console.log(completion.choices[0].message.content);
```

### System Instructions

Guide the model's behavior with system messages:

```javascript
const completion = await lexa.chat({
  messages: [
    { 
      role: 'system', 
      content: 'You are a helpful assistant that speaks like a pirate.' 
    },
    { 
      role: 'user', 
      content: 'What is the weather like today?' 
    }
  ],
  model: 'lexa-x1',
  temperature: 0.9
});

console.log(completion.choices[0].message.content);
```

### Multi-turn Conversations

Maintain context across multiple exchanges:

```javascript
const messages = [
  { role: 'system', content: 'You are a helpful coding assistant.' },
  { role: 'user', content: 'What is a closure in JavaScript?' }
];

const response1 = await lexa.chat({
  messages,
  model: 'lexa-rho',
  temperature: 0.7
});

// Add the assistant's response to the conversation
messages.push({
  role: 'assistant',
  content: response1.choices[0].message.content
});

// Continue the conversation
messages.push({
  role: 'user',
  content: 'Can you show me an example?'
});

const response2 = await lexa.chat({
  messages,
  model: 'lexa-rho'
});

console.log(response2.choices[0].message.content);
```

### Streaming Responses

Get real-time streaming output for long responses:

```javascript
const completion = await lexa.chat({
  messages: [
    { role: 'user', content: 'Write a short poem about coding.' }
  ],
  model: 'lexa-rho',
  temperature: 0.8,
  stream: true
});

// Handle streaming response
const reader = completion.stream.getReader();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;

  if (value.type === 'text-delta') {
    process.stdout.write(value.delta);
  } else if (value.type === 'finish') {
    console.log('\n\nUsage:', value.usage);
  }
}
```

<Note>
Streaming is perfect for chatbots and interactive applications where you want to display responses as they're generated.
</Note>

## Configuration Options

Customize your Lexa client with additional configuration:

```javascript
const lexa = new Lexa(process.env.LEXA_API_KEY, {
  baseURL: 'https://www.lexa.chat/api', // Custom base URL (default shown)
});
```

### Chat Completion Options

| Parameter | Type | Description |
|-----------|------|-------------|
| `messages` | `Array` | Array of message objects with `role` and `content` |
| `model` | `string` | Model ID (default: `'lexa-mml'`) |
| `temperature` | `number` | Controls randomness (0-2, default: 1.0) |
| `max_tokens` | `number` | Maximum tokens to generate |
| `stream` | `boolean` | Enable streaming response (default: `false`) |

## Understanding Responses

A typical response structure:

```javascript
{
  choices: [{
    message: {
      role: 'assistant',
      content: 'The response text from the model'
    },
    finish_reason: 'stop' // or 'length', 'content_filter'
  }],
  usage: {
    prompt_tokens: 15,
    completion_tokens: 42,
    total_tokens: 57
  }
}
```

## Error Handling

Implement proper error handling for production applications:

```javascript
try {
  const completion = await lexa.chat({
    messages: [
      { role: 'user', content: 'Hello!' }
    ],
    model: 'lexa-mml'
  });
  
  console.log(completion.choices[0].message.content);
} catch (error) {
  if (error.status === 401) {
    console.error('Invalid API key');
  } else if (error.status === 429) {
    console.error('Rate limit exceeded');
  } else {
    console.error('Error:', error.message);
  }
}
```

## Complete Example

Here's a complete example that puts it all together:

```javascript app.js
import Lexa from '@robilabs/lexa';

async function main() {
  // Initialize client
  const lexa = new Lexa(process.env.LEXA_API_KEY);

  // Example 1: Simple chat
  console.log('Simple Chat:');
  const simple = await lexa.chat({
    messages: [{ role: 'user', content: 'Hello! Can you tell me a joke?' }],
    model: 'lexa-mml',
    temperature: 0.7,
    max_tokens: 100
  });
  console.log(simple.choices[0].message.content);
  console.log('Usage:', simple.usage);

  // Example 2: System message
  console.log('\nWith System Message:');
  const withSystem = await lexa.chat({
    messages: [
      { role: 'system', content: 'You are a helpful assistant that speaks like a pirate.' },
      { role: 'user', content: 'What is the weather like today?' }
    ],
    model: 'lexa-x1',
    temperature: 0.9
  });
  console.log(withSystem.choices[0].message.content);

  // Example 3: Streaming
  console.log('\nStreaming Response:');
  const streaming = await lexa.chat({
    messages: [{ role: 'user', content: 'Write a short poem about coding.' }],
    model: 'lexa-rho',
    temperature: 0.8,
    stream: true
  });

  const reader = streaming.stream.getReader();
  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    if (value.type === 'text-delta') {
      process.stdout.write(value.delta);
    } else if (value.type === 'finish') {
      console.log('\n\nUsage:', value.usage);
    }
  }

  // Example 4: List models
  console.log('\nAvailable Models:');
  const models = await lexa.models();
  models.data.forEach(model => {
    console.log(`- ${model.id} (${model.owned_by})`);
  });
}

main().catch(console.error);
```

```javascript OpenAI
  messages: [{ role: 'user', content: 'Hello!' }],
  model: 'gpt-3.5-turbo'
});
```

```javascript Lexa
import Lexa from '@robilabs/lexa';

const lexa = new Lexa('your-key');
```
